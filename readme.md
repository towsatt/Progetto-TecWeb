Repo per il progetto di Tecnologie Web

Il progetto consiste in un sito per la gestione delle campagne di D&D.
Fornisce la possibilità di condividere progressi con giocatori di tutto il mondo

Per testare in locale scaricare:
- [Docker Desktop](https://www.docker.com/products/docker-desktop/).
- Estensione di VS Code: **Dev Containers**

Poi:
- Fare il clone/pull della repo
- Premere `Ctrl + Shift + P`
- Selezionare **`Dev Containers: Reopen in Container`**.

Per testare file php:
- php sito/PHP/test_research_user.php
